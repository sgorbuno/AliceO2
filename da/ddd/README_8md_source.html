13 ### Scope
14 The ALICE O2 software repository contains the framework, as well as the detector specific, code for the reconstruction, calibration and simulation for the ALICE experiment at CERN for Run 3 and 4. It also encompasses the commonalities such as the data format, and the global algorithms like the global tracking.
15 Other repositories in AliceO2Group contain a number of large common modules, for instance for Monitoring or Configuration.

45 ### Formatting
46 The project uses `clang-format` to push for a common code formatting according
47 the the `clang-format` configuration files in this repository. With an adiabatic
48 approach, all changes have to follow the formatting rules. A tool script can be
49 used to integrate the formatting into `git` and suggest formatting only for
50 changed lines.
51 
52 ##### Install `clang-format`
53 Use alienv to load `clang` which includes `clang-format` : `alienv load Clang/latest`
54 If Clang is not yet build, use alibuild : `aliBuild build --defaults o2 Clang`
55 If you use your own clang installation, make sure you have at least version 3.9.
56 
57 ##### Install `clang-format` git integration
58 The `git-clang-format` Python script integrates `clang-format` into `git`.
59 Put it somewhere in your path and ensure that it is executable, e.g.
60 
61 ~~~~~~~{.sh}
62 
63 cd $HOME
64 mkdir -p bin
65 cd bin
66 wget llvm.org/svn/llvm-project/cfe/trunk/tools/clang-format/git-clang-format
67 chmod u+x git-clang-format
68 ~~~~~~~
69 
70 Note: installation of the script will be added to build of AliceO2 software stack.
71 
72 ##### Checking formatting
73 Now, `git clang-format` will invoke `clang-format` on the changes in current files
74 or a specific commit. E.g. for the last commit
75 ~~~~~~~
76 git clang-format HEAD~1
77 ~~~~~~~
78 
79 Or for all commits done with respect to the remote branch state
80 ~~~~~~~
81 git clang-format origin/dev
82 ~~~~~~~
83 
84 ##### Checking files
85 Show updated version of complete file :
86 ~~~~~~~
87 clang-format -style=file SOURCEFILE
88 ~~~~~~~
89 
90 Directly apply the style to file :
91 ~~~~~~~
92 clang-format -style=file -i SOURCEFILE
93 ~~~~~~~
